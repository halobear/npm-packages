## @halobear/vue-upload

vue3.0 上传组件封装(如果是 vue2.0，请使用 @halobear/vue-upload@1.1.4)

## Install

```bash
yarn add @halobear/vue-upload
```

## Usage

```js
import Upload from '@halobear/vue-upload'
import '@halobear/vue-upload/lib/style.css'
```
