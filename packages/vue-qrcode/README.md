## @halobear/vue-qrcode

> qrcode 的 vue 组件

## 安装

```bash
yarn add @halobear/vue-qrcode
```

## 使用

```js
import Qrcode from "@halobear/vue-qrcode";

export default {
  components: {
    Qrcode
  }
};
```

```html
<div>
  <qrcode :width="100" :height="100" />
</div>
```
